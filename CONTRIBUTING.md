# Contributing

Thanks for your interest in contributing to `spydus-cli`.

## Setup

- `uv sync`
- `uv run poe check`

## Development workflow

1. Add or update tests for your change.
2. Keep JSON output stable for machine consumers.
3. Avoid logging credentials, session cookies, or personal account data.
4. Run lint + tests before opening a PR.

## Pull requests

- Include a concise summary and motivation.
- Mention any schema/CLI changes clearly.
- Add docs updates when behavior changes.
